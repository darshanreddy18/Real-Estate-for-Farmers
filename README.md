# 🏡 Real Estate for Farmers

## 📖 About
Real Estate for Farmers is an Android and Web-based application that helps farmers search, list, and manage agricultural land and properties.

## ✨ Features
- Farmer Registration & Login
- Property Listing
- Property Search
- Admin Dashboard
- Secure Authentication
- Database Integration

## 🛠️ Technologies Used
- Android Studio
- Java
- ASP.NET
- HTML
- CSS
- JavaScript
- SQL Database

## 📂 Project Structure
Apps/
Web/
schema.sql

## 🚀 How to Run

### Android App
1. Open the Apps folder in Android Studio.
2. Build and run the project.

### Web Application
1. Open the Web project in Visual Studio.
2. Configure the database.
3. Run the application.

### Database
Import `schema.sql` into your SQL database.
